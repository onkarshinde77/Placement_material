## Some imp commands
-> build docker image : docker build -t repo-name .
-> check docker image : docker image
-> delete docker image : docker rmi repo-name
-> remove partially created docker image : docker system prune -f
-> if you want to delete all dependency then run this commands
	- pip freeze > uninstall.txt
	- pip uninstall -y -r uninstall.txt
	- del uninstall.txt
-> to check all list of dependency : pip list
-> list of all current running container : docker ps
-> to stop running port of docker : docker stop id
